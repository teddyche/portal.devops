Valeurs FQDN par défaut

Ces valeurs pré-remplissent les champs FQDN dans le builder. L'utilisateur peut les modifier avant de générer.

Middlewares

Gérez les middlewares disponibles dans le builder. Activé = visible et sélectionnable. Bientôt = affiché mais non cliquable. Désactivé = masqué.

Autres rôles

Ces rôles sont inclus dans le ZIP comme squelettes Ansible mais ne sont jamais appelés dans les playbooks générés. Utiles pour fournir des exemples ou des utilitaires.
